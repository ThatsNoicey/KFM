// User Schema (server/models/User.js)
{
  username: String,
  email: String,
  passwordHash: String,
  role: { type: String, enum: ['patient', 'physio'] },
  assignedPatients: [ObjectId], // Only if physio
  createdAt: Date
}

// RehabPlan Schema (assigned by physio)
{
  patientId: ObjectId,
  weekStart: Date,
  exercises: [{ day: String, exercise: String, sets: Number, reps: Number }]
}

// Log Schema (submitted by patient)
{
  patientId: ObjectId,
  date: Date,
  completed: Boolean,
  painLevel: Number,
  rangeOfMotion: Number,
  notes: String
}

// Feedback Schema
{
  physioId: ObjectId,
  patientId: ObjectId,
  message: String,
  date: Date
}

# Next-Gen Physio App

A role-based rehabilitation tracker that allows patients to log progress and physios to monitor trends, assign plans, and communicate feedback.

## Features
- Secure login for patient & physio
- Rehab plan assignment
- Daily logs of ROM, pain, completion
- Graphical feedback and progress insights
- Future: wearable emulation, real-time messaging

## Setup
- `cd client && npm install`
- `cd server && npm install`
- Use `concurrently` to run both in dev mode

```bash
npm install concurrently
npm run dev
```

## Tech Stack
- Frontend: React + Tailwind
- Backend: Express + MongoDB
- Auth: JWT + Bcrypt
- Charts: Recharts
- Dev Tools: ESLint, Prettier, dotenv
